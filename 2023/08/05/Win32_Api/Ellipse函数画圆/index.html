Ellipse函数画圆

Ellipse 函数 (wingdi.h)

Ellipse函数绘制一个椭圆。 椭圆的中心是指定边界矩形的中心。 椭圆形是使用当前笔轮廓的，并使用当前画笔填充。

语法

参数

[in] hdc

设备上下文的句柄。

[in] left

边界矩形左上角的 x 坐标（以逻辑坐标表示）。

[in] top

边界矩形左上角的 y 坐标（以逻辑坐标表示）。

[in] right

边界矩形右下角的 x 坐标（以逻辑坐标表示）。

[in] bottom

边界矩形右下角的 y 坐标（以逻辑坐标表示）。

返回值

如果该函数成功，则返回值为非零值。

如果函数失败，则返回值为零。

注解

Ellipse 既不使用也不更新当前位置。

Windows 95和Windows 98：限定矩形的坐标值之和不能超过32767。nLeftRect与nRigthRect或nTopRectn与BottomRect之和不能超过32767

注意：

Ellipse(x1,y1,x2,y2),你必须确保x2-x1==y2-y1,这样你画出的就是正圆,而不是椭圆了.
